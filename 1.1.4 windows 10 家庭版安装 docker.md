## 二 如何查看自己的电脑是否支持虚拟化

```
在 cmd 里执行 systeminfo


Hyper-V 要求:     虚拟机监视器模式扩展: 是
                  固件中已启用虚拟化: 否
                  二级地址转换: 是
                  数据执行保护可用: 是
```
