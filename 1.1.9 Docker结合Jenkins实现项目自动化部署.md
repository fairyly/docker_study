# Docker结合Jenkins实现项目自动化部署

## 1.下载Jenkins镜像

```
docker pull jenkins
默认情况下，如果没有指定版本号和前缀的话，默认都是到Docker Hub官方去下载镜像的，默认版本标签是“latest”
```



## 2.启动Jenkins容器
```
docker run -d -p 28080:8080 --name jenkins -v /opt/jenkins:/var/jenkins_home jenkins

```
